# Visual Studio Code extension for Ansible

## Features

- run ansible from Visual Studio Code
  - On Windows, run ansible inside docker
  - On Non-windows platform, run ansible locally

## Requirements

|platform|prequisites|
|--------|-----------|
|Windows|docker|
